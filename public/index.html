Go/Golang Server Homepage

  • GitHub repository hosted at: https://github.com/wizzard262/golang-server/
  • README.md (full Go setup and hosting instructions) README.md
  • Hosted on Vercel, project at: https://vercel.com/jones-steve-projects/golang-server
  • Build - Vercel automatically builds upon GIT repo update

ENDPOINTS
  • STATIC - ROOT index.html HTML homepage (this page with CSS & JS).
  • DYNAMIC - /api/hello Text from within the Go/Golang server.
  • DYNAMIC - /api/weather?lat=53.4808&lon=-2.2426 Weather from OpenMeteo API and JSON formatted.
    (Manchester UK Coordinates)

Things Vercel project can do (Go + static):
  • Use environment variables
    Store API keys or secrets in Vercel → access via os.Getenv().
  • Add redirects & rewrites
    Control routing with vercel.json (SPA fallback, API routing, etc).
  • Dynamic API routes
    Use folders like /api/user/[id].go for parameterised endpoints.

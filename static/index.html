Go/Golang Server Homepage

  • GitHub repository hosted at: https://github.com/wizzard262/golang-server/
  • README.md (full Go setup and hosting instructions) README.md
  • Hosted on Vercel, project at: https://vercel.com/jones-steve-projects/golang-server
  • https://golang-server-beta.vercel.app/ shows the content of the static HTML homepage (this page).
  • https://golang-server-beta.vercel.app/api/hello shows dynamically generated text from within the Go/Golang server.


Things Vercel project can do (Go + static):
  • Multiple Go API endpoint
    Add more files under /api like weather.go, user.go, data.go.
  • Return JSON from Go
    Build proper API responses for your frontend.
  • Serve a full static frontend
    Put HTML/CSS/JS in /static and call your Go API from it.
  • Use environment variables
    Store API keys or secrets in Vercel → access via os.Getenv().
  • Add redirects & rewrites
    Control routing with vercel.json (SPA fallback, API routing, etc).
  • Dynamic API routes
    Use folders like /api/user/[id].go for parameterised endpoints.
  • Call external APIs
    Use Go to fetch weather, GitHub, or any REST API.
  • Build a micro‑API
    Multiple Go serverless functions acting as small services.
